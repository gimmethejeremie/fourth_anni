# Character Art

Place final visual-novel character art here and reference it from `src/data/guides.ts` with public paths such as:

```ts
artSrc: "/characters/kagura.png"
```

Recommended filenames:

- `kagura.png`
- `imed.png`
- `lil-wayne.png`
- `kuro.png`
- `nova.png`
- `stone.png`

Transparent PNG or WebP files work best because the visual-novel frame uses `object-fit: contain`.
